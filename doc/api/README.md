# CurryBu API

CurryBu では、API を用意しています。  
認証を必要としない API については自由にアクセスできます。

## 認証
Requires Authentication が Yes となっている箇所は OAuth 認証が必要です。

## 画像のパス
production の場合、画像のパスは S3 の URL が返ってきます。  
ドキュメントの方はあとで対応します。
